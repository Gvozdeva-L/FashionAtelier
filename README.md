## Опис

Десктопний застосунок для автоматизації роботи ательє, написаний мовою C#

Програма дозволяє додавати нові замовлення, переглядати список замовлень, шукати замовлення за різними критеріями, редагувати та видаляти інформацію, працювати з базою даних SQLite для збереження даних.

Під час розробки були використані принципи об'єктно-орієнтованого програмування, класи, робота з файлами та базою даних.

## Технології
- C#
- DBeaver(SQLite)
- Windows Forms
- Visual Studio 2022
  
## Функціонал
- Додавання, перегляд, редагування та видалення замовлень (CRUD)
- Фільтрація замовлень за статусом, майстром та послугою
- Валідація введених даних
- Робота з реляційними зв'язками (багато-до-багатьох)
  
## Про проєкт

Цей проєкт був створений як курсова робота з метою закріплення знань із програмування на C#. Основна увага приділялась роботі з класами, взаємодії з базою даних та реалізації основних операцій для роботи із замовленнями ательє.
